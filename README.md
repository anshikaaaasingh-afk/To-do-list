# ✅ To-Do List App

A clean, beginner-friendly To-Do List web app built with HTML, CSS, and JavaScript.

## 🚀 Features

- Add tasks by typing and pressing **Enter** or clicking **Add**
- Mark tasks as done with a checkbox
- Delete individual tasks
- Filter tasks by **All**, **Active**, or **Done**
- Clear all completed tasks at once
- Tasks are saved in **localStorage** — they persist after you close the tab
- Fully responsive — works on mobile and desktop

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla — no frameworks or libraries)

## 📁 Project Structure

```
todo-list/
├── index.html   → Page structure
├── style.css    → Styles and layout
├── script.js    → App logic
└── README.md    → This file
```

## 🏃 How to Run

1. Clone or download this repository
2. Open `index.html` in your browser
3. That's it — no install needed!

```bash
git clone https://github.com/YOUR_USERNAME/todo-list.git
cd todo-list
open index.html
```

## 💡 What I Learned

- Storing and retrieving data with `localStorage`
- Filtering arrays with `.filter()` and updating UI dynamically
- Keeping HTML, CSS, and JS in separate files for clean code structure
